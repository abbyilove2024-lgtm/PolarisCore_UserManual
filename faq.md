# 常见问题（FAQ）

## Q1：使用过程中遇到报错或功能建议，如何反馈？

你可以通过两种方式提交问题：

1. 软件内置的 **「我要反馈」** 入口直接提交。
2. 通过官方TG支持渠道联系产品支持团队。

产研团队会定期评估和处理，并将常见问题与改进结果同步到本手册。

---

## Q2：Polaris Core 与其他工具（如外部 OpenClaw 或 Claude Code）有什么区别？

Polaris Core 是 **针对集团办公场景深度优化的自研平台**，相比通用工具有以下差异：

| 维度 | 紫微Agent (Polaris Core) | 通用 AI 工具 |
| --- | --- | --- |
| Skill 管理 | 内置易用的 Skill 工作台与技能广场 | 通常需要写代码或手动配置 |
| 本地执行 | 强本地文件执行能力，可直接处理桌面文件 | 多为云端沙盒，受限较多 |
| 办公场景适配 | 针对集团内常见办公流程做了深度优化 | 通用场景，需自行调优 |
| 模型策略 | 智能匹配 + 可手动切换底层大模型 | 通常绑定单一模型 |

---

## Q3：邀请码从哪里获得？

邀请码由官方统一发放。如果你尚未持有，请联系你的 **服务支持人员** 或所在组织的对接人。

---

## Q4：Skill 安装失败怎么办？

请按顺序排查：

1. 确认 `.md` 文件没有损坏，可以用文本编辑器打开看到完整内容。
2. 确认 Skill 的依赖（例如外部 API key）已经配置。
3. 在 **「已安装」** 列表中尝试卸载后重装。
4. 仍无法解决，使用 **「我要反馈」** 联系支持团队，并附上 Skill 文件。
